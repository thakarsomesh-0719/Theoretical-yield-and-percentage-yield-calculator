# Theoretical-yield-and-percentage-yield-calculator
Calculates Theoretical yield and percentage yield of any single step and single product chemistry synthesis on the basis of stoichiometry
Inputs : molar masses, coefficients, mass or volume of limiting reagent, density if volume
Assumptions: user has already balanced the equation and identified the limiting reagent — the script does not do this
Limitations: single product only, no multi-step reaction support
